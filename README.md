# shooking
슈킹 신발 쇼핑몰 개발을 위한 문서 가이드 저장소입니다.
구현해야 할 기능
1. 모바일 UI : 웹페이지를 기본으로 하되 모바일 환경에서도 사용이 편리하도록
2. 상품 사진 : 상품 목록에서 상품 사진이 잘 보이도록
3. 장바구니에 담긴 상품 갯수 실시간 업데이트
4.  배포 URL : 작업한 결과물이 직원들이 확인해 볼 수 있도록
